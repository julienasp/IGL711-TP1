Detailed Description

Fichier d'initialisation, wordpress repère dynamiquement ce fichier nous permettant ainsi d'activer ou desactiver le plugin.

Plugin Name: Plugin de gestion d'anomalies Description: Mini gestionnaires d'anomalies pour le Travail pratique 1 du cours IGL711 License: éducation Version: 0.0.1 Author: Julien Aspirot

Fichier d'initialisation, wordpress repère dynamiquement ce fichier nous permettant ainsi d'activer ou desactiver le plugin. Lorsque Wordpress active ou desactive le plugin ce fichier sera executé.
